Наши технологии

HTML

HyperText Markup Language

HTML5, for create the framework of the future web site, use pug template engine and preprocessor.

CSS

Cascading Style Sheets

CSS3, flexbox, grid, creating the web site wireframe appearance using preprocessors sass, scss, less, stylus.

JavaScript

Programming language

Dynamic site, working with the server by sending requests, implementing all the functional logic of the client part.

Jquery

JavaScript library

Simplifies the work with the DOM tree, easy and convenient syntax, recently losing popularity.

Bootstrap

Open-source CSS framework

Quickly create interfaces in a short time using HTML + CSS + Jquery. Open source editable.

Git

Development Versioning

Data loss protection, convenient interaction console, command program. Workflow design.

React

JavaScript framework

Single page application convenient routing, the best work with the DOM tree.

Redux

JavaScript library

Managing the state of the application by creating a store and interacting with it through actions.

Next

Open source web framework

Server rendering to improve seo optimization. Enhanced css-in-js features.

Gulp

Project collector

Allows you to collect all the files in while compressing them to minimum values ​​and automatic.

Material-ui

Quantum Paper

Support ready-made solutions for complex and custom components, easily change component styles.

Styled components

JS customize library

A comfortable technique for customizing javascript components while not loading the system.

Photoshop

Graphics editor

The most popular editor for working with web design, has a large number of plugins.

Figma

Cascading Style Sheets

Has a built-in css interpreter, this greatly speeds up the development of cascading styles.

Sketch

Graphics editor

Graphic editor for design development in the MAC system, which makes it unique, but not cross-platform.

Avocode

Graphics editor

The most convenient and best layout helper in my opinion. Has a built-in interpreter for all graphic editors.
